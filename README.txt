Sistema de Gerenciamento de Biblioteca - Trabalho Final

Como executar:
1. Certifique-se de que todos os arquivos estão na mesma pasta.
2. Execute o arquivo main.py com Python 3:
   python main.py

Requisitos:
- Nenhuma biblioteca externa é necessária.
