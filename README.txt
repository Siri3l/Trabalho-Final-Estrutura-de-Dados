===========================================
  SISTEMA DE GERENCIAMENTO DE BIBLIOTECA
===========================================

Descrição:
-----------
Este é um sistema simples de gerenciamento de biblioteca, desenvolvido como Trabalho Final da disciplina de Estrutura de Dados. 
A aplicação permite cadastrar livros e usuários, realizar empréstimos, devoluções, buscas e outras operações.

Funcionalidades:
-----------------
- Cadastro de livros e usuários
- Empréstimo e devolução de livros
- Remoção de livros (caso não estejam emprestados)
- Remoção de usuários (caso não possuam livros emprestados)
- Busca de livros por título, autor ou ISBN
- Listagem de livros disponíveis e emprestados

Como executar:
---------------
1. Certifique-se de que todos os arquivos do projeto estejam na mesma pasta.
2. Execute o arquivo main.py com Python 3 utilizando o comando:

   python main.py

Observação:
------------
Nenhuma biblioteca externa é necessária para rodar este projeto.
